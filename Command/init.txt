====== init ======

copy file to [[/etc/init.d/]]
update-rc.d xxx defaults NN

$ sudo chmod 755 [[/etc/init.d/test]]
$ cd /etc/init.d
$ sudo update-rc.d test defaults 95

$ cd /etc/init.d
$ sudo update-rc.d -f test remove
